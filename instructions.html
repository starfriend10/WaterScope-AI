WaterScope-AI MCQA Demo Instructions

Getting Started

Welcome to the WaterScope-AI Multiple Choice Question Answering (MCQA) demo. This tool allows you to compare the performance of two specialized language models on water sustainability questions.

1

Load a Question

You can either:

  • Type your own question and options in the input fields
  • Click on any row in the example data table to automatically populate the form
2

Add Options (Optional)

By default, 4 options (A-D) are available. You can add up to 8 options total using the "Add Option" button.

3

Generate Explanation (Optional)

Check the "Generate Explanation" box if you want the models to provide reasoning for their answers.

4

Run Analysis

Click the "Run Analysis" button to process your question through both models. The system will display:

  • Predicted answer letter (A, B, C, etc.)
  • Raw model output
  • Processing time

Model Information

DA-DPO Model

Preference-Optimized

This model has been fine-tuned using Direct Preference Optimization (DPO) to align with human preferences for water sustainability content.

DA-IT Model

Instruction-Tuned

This model has been instruction-tuned on a diverse set of water sustainability questions and answers to improve its performance on domain-specific tasks.

Interpreting Results

Prediction Letter

The letter (A, B, C, etc.) represents the model's chosen answer from the options provided.

Raw Output

The complete text generated by the model. This may include explanations if the option was selected.

Processing Time

The time taken to generate the response. This can vary based on question complexity and server load.

Tips for Best Results

Use clear, specific questions related to water sustainability
Provide distinct options for better model discrimination
Select "Generate Explanation" for insight into model reasoning
Use the example data table for quick testing

Troubleshooting

API Connection Issues

If the API status shows connection problems, try refreshing the page. The system will automatically attempt to reconnect.

No Response from Models

Ensure you've provided both a question and at least two options before running the analysis.

Clear Form

Use the "Clear" button to reset the input fields without affecting ongoing processing.
